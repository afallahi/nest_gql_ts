## Description

[Nest](https://github.com/nestjs/nest) project with Typescript, GraphQL, and connected to MongoDB.

## Stack


- NestJS
- Typescript
- GraphQL
- MongoDB


## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev
```
